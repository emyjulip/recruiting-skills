# Recruiting Skills

A small repository of practical recruiting skill notes and templates you can reuse and expand.

Contents
- `sourcing.md` — sourcing strategies, boolean/X-ray search examples, outreach templates
- `interviewing.md` — structured interviewing, scoring rubrics, question banks
- `negotiation.md` — offer negotiation guidance, equity and salary tips
- `onboarding.md` — 30/60/90 plans, manager handoffs, ramp success metrics
- `employer-branding.md` — career site and candidate experience tips
- `pipeline-health-check.md` — pipeline health check: funnel metrics, velocity, stage conversion, and aging candidates

Getting started
1. Clone the repo locally (or work inside the `recruiting-skills` folder you already created):

```bash
git clone https://github.com/YOUR-USERNAME/recruiting-skills.git
cd recruiting-skills
```

2. Make changes or add new skill notes. Commit locally:

```bash
git add .
git commit -m "Improve README and add templates"
```

Pushing to GitHub (if you haven't already added the remote)
Replace the URL below with your repository URL, then run:

```bash
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/recruiting-skills.git
git push -u origin main
```

How to contribute
- Open issues for new skill ideas or templates you want.
- Submit pull requests with improvements, new templates, or example workflows.

Suggested project structure
- Keep each skill as a standalone markdown file.
- Add subfolders for templates (e.g., `templates/offer-letters.md`) or examples (`examples/`).

License & contact
This repo contains short notes and templates; add a LICENSE file if you want an explicit license. For questions or collaboration, open an issue in the repository.

Enjoy building your recruiting playbook.
